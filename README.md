# ESO БГ — Батерии в Диаграмата

Разширение за Chrome, което показва скритото разреждане на батериите (ССЕЕ) в диаграмата за генерация на [eso.bg](https://www.eso.bg/doc/?460).

## Инсталация

1. Изтеглете и разархивирайте това репо
2. Отворете `chrome://extensions`
3. Включете **Режим за разработчици** (горе вдясно)
4. Натиснете **Зареди разпакетирано** → изберете папката
5. Отворете [eso.bg/doc/?460](https://www.eso.bg/doc/?460)

Тъмнозеленият сектор се появява автоматично. Разширението не изпраща никакви данни.
